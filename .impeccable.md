## Design Context

### Users
AI agent builders who need Israeli grocery shopping plans for agents — without decoding API schemas first.

### Brand Personality
Friendly-operational. Wolt Market clarity: airy Hebrew landing, one loud CTA, grocery visuals. Super MCP truth: asks before uncertain swaps.

### Aesthetic Direction
- Hebrew RTL; Wolt cyan palette (darker CTA for AA contrast)
- Story first, technical proof second
- One basket example; three human benefits; collapsed developer details
- Large grocery photography; sparse cards; pill CTAs with RTL `<<`

### Design Principles
1. Five-second comprehension: רשימה נכנסת → תוכנית קנייה יוצאת
2. No API field names above the developer disclosure
3. One dominant CTA: בקשו גישה
4. Progressive disclosure for MCP/REST tooling
5. Prefer concrete Hebrew outcomes over infrastructure jargon
